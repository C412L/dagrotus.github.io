Overview

Lor Deck Name Editor provides various tools that help customizing deck names in Legends of Runeterra, including: text formatting options, text coloring and sprite insertion.

Text formatting

Text formatting is assigned using following tags:

  • <b> - text formatted Bold;
  • <i> - text formatted Italic;
  • <u> - text formatted Underlined;
  • <s> - text formatted with line through it;
  • <sub> - text formatted as subscript
  • <sup> - text formatted as superscript
  • <mark> - text formatted Marked;

Tags can be written manually or by using buttons to apply formatting automatically. Put opening tag before text, to apply format. Closing tag is not required, but can be used to apply formatting between tags for example: "<b>Bold text here" is the same as "<b>Bold text here</b>" but not the same as "<b>Bold text here</b> regular text here". Selecting text and pressing format button will apply format to selection. If text is not selected, tag will be inserted at cursor position.

Note: in game underline and line-through format will be applied to sprites, but preview here does not display it.

Text coloring

Text color is defined by hexadecimal color code or a color name inside the color tag. Text coloring can be assigned by following types of tags:

  • Short color code tag (<#0F0> or with opacity <#0F0A>);
  • Full color code tag (<#00FF00> or with opacity <#00FF00AA>);
  • Color tag with specified color code, short or full (<color=#0F0> or <color=#00FF00>);
  • Color tag with color name (<color=green>), following color names can be used: red, orange, yellow, green, blue, purple, black and white;

Like text format, tags can be written manually or by using color picker tool to apply color automatically. Closing tag (</color>) is not required, but can be used to apply coloring between tags. In game color will be applied to sprites, but preview here might display it incorrectly in Firefox and some other browsers. When picking color with color picker tool, you can choose to apply short or full color code.

Sprites

Sprites can be inserted with following tags:

  • Numbered sprite tag (<sprite=1>). Number can be between 1 and 24;
  • Named sprite tag (<#sprite name=Noxus>);

Sprites can be inserted with sprite palette, it will always choose the shortest available sprite tag.
